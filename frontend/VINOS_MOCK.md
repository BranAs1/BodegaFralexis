## 🎯 Casos de Uso de los Mocks

### Pruebas de Filtros
```javascript
// Filtrar por tipo "Tinto"
const tintos = vinosMock.filter(v => v.tipoVino === "Tinto");
// Resultado: 5 vinos

// Filtrar por rango de precio
const midrango = vinosMock.filter(v => v.precioMinorista >= 300 && v.precioMinorista <= 400);
// Resultado: 6 vinos

// Filtrar ofertas
const ofertas = vinosMock.filter(v => v.esOferta);
// Resultado: 5 vinos
```

### Pruebas de Stock
```javascript
// Vinos con bajo stock
const bajoStock = vinosMock.filter(v => v.stock < 20);
// Resultado: 2 vinos (Pinot Noir, Tannat)

// Vinos sin stock
const agotados = vinosMock.filter(v => v.stock === 0);
// Resultado: 0 vinos (todos disponibles)
```

### Pruebas de Precios
```javascript
// Calcular descuento mayorista (6+ botellas)
const vino = vinosMock[0]; // Malbec
const precioMin3 = vino.obtenerPrecio(3);  // $350 (minorista)
const precioMin10 = vino.obtenerPrecio(10); // $280 (mayorista)
const ahorro = (350 - 280) * 10; // $700 de ahorro
```

## 💾 Cómo se Cargan

### Escenario 1: API Disponible
```
1. Se intenta fetch a http://localhost:3000/api/vinos
2. ✅ Éxito → Se cargan datos reales de la base de datos
3. Notificación: "✓ X vinos cargados desde API"
```

### Escenario 2: API No Disponible
```
1. Se intenta fetch a http://localhost:3000/api/vinos
2. ❌ Error o timeout
3. Se cargan automáticamente los mocks
4. Notificación: "✓ 12 vinos cargados (Modo Demostración)"
```

## 🔄 Cómo Usar las Clases OOP

### Crear una instancia de Vino
```javascript
import { Vino } from './Vino.js';

const miVino = new Vino(
  1,                          // id
  "Malbec Reserva",          // nombre
  "Finca Fralexis",          // marca
  350,                       // precioMinorista
  280,                       // precioMayorista
  45,                        // stock
  "Malbec",                  // tipoUva
  "Tinto",                   // tipoVino
  2020,                      // anoCosecha
  750,                       // tamanoMl
  true                       // esOferta
);

// Usar métodos
console.log(miVino.obtenerDescripcion()); // "Malbec Reserva - Finca Fralexis (2020)"
console.log(miVino.estaDisponible());     // true
console.log(miVino.obtenerPrecio(10));    // 280 (mayorista)
```
